If you already have a html page you  can just copy it here.  If not..

Open a composer window ( Windows ->  Composer  these are at the top of your browser page)

Start making your page.

Make sure you save with a descriptive name, and save often you don't want to loose work!

You can inspect your artistry in a browser URL window type:

file:///home/siggy/docs/Adding-a-Page-HOWTO.html  (This is the name of this page as I was working on it)

When you are done you can copy it to:

/web/hollywood/technical_services/pages/CS-NOTES OR wherever you think it should reside.

# chown www.web <Your-Page-Name-Here>.html

Next you will have to edit the "index.html" so that all your hard work will be available.
