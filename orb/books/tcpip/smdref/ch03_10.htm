3.10 Hints

-f and -N

Must match between the configuration file's K line and the command-line run to execute makemap(1). But note that in most cases you probably don't want to use either switch.

text class

Lookups on heavily loaded systems can be very slow. They can also be slow when the text file is extremely large. In either instance try to convert to an ndbm- or db-style database.
