Chapter 2
UNIX Commands
 

uulog

uulog [options]

Print information from the uucp or uuxqt log files, which reside in /var/uucp/.Log (down subdirectories uucico or uuxqt). See also tail.

Options

-fsys

Issue a tail -f to print the most recent actions for a given system.

-ssys

Print all actions for the given system.

-uuser

Print commands issued by user (SVR3 only).

-x

Check the uuxqt log file for the given system (used with -f or -s).

-n

Execute a tail command of n lines (used with -f).
