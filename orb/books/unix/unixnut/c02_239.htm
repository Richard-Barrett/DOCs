Chapter 2
UNIX Commands
 

uupick

uupick [option]

Query the status of files sent to the user with uuto.

Option

-ssystem

Search only for files sent from system.

Interactive responses

a[dir]

Move all files sent from system to the named directory.

d

Delete the entry.

m[dir]

Move the file to the directory dir.

p

Print the file.

q

Quit uupick.

*

Print a command summary.

!cmd

Execute the shell command cmd.

EOF

Quit uupick.

RETURN

Move to next entry.
