UNIX Commands
 

vacation

vacation [options]

Automatically return a mail message to sender announcing that you are on vacation. To disable this feature, type mail -F "".

Options

-d

Append the date to logfile.

-F user

Forward mail to user when unable to send mail to mailfile.

-l logfile

Record in logfile the names of senders who received an automated reply (default is $HOME/.maillog).

-m mailfile

Save received messages in mailfile (default is $HOME/.mailfile).

-M msg_file

Use msg_file as the reply to mail automatically (default is /usr/lib/mail/std_vac_msg).
