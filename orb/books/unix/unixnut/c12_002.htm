Nroff and Troff
 

.ad

.ad [c]

Adjust output lines according to format c. Fill mode must be on (see .fi). With no argument, return to previous adjustment. The current adjustment mode is stored in register .j, with the following values: 0=l, 1=b, 3=c, 5=r (see .na).

Values for c

b

Lines are justified.

n

Lines are justified.

c

Lines are centered.

l

Lines are flush left.

r

Lines are flush right.
