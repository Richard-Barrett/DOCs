Chapter 12
Nroff and Troff
 

.ta

.ta n[t] [+]m[t] ...

Set tab stops at positions n, m, etc. If t is not given, tab is left-adjusting. Use a + to move relative to the previous tab stop. The \t escape is similar.

Values for t

L

Left adjust.

R

Right adjust.

C

Center.
