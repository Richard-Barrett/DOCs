.LI

.LI [mark] [1]
text

Item in list. List must be initialized (see .AL, .BL, .DL, .LB, .ML, and .VL) and then closed using .LE. If mark is specified, it replaces mark set by list-initialization macro. If mark is specified along with second argument of 1, then mark is prefixed to current mark.
